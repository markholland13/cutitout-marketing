Delivery and Returns

delivery, transit issues, and returns for custom-made parts

This page explains where orders can be delivered, how shipping is charged, and what customers should do if an order arrives damaged, incomplete, or incorrect.

Delivery Summary
Orders are restricted to UK mainland delivery addresses
Shipping is calculated automatically during checkout
Custom-made parts are not treated like standard retail stock
Transit issues should be reported promptly with order details

Delivery area

We currently accept orders for UK mainland delivery addresses only. Non-mainland, overseas, and unsupported addresses may require manual review or may not be accepted through the online checkout.

Shipping charges

Shipping charges are calculated automatically based on the order weight and package dimensions. The delivery charge shown at checkout forms part of the order total.

Transit issues

If a package arrives damaged, incomplete, or appears to have been lost in transit, customers should contact us promptly with the order details and any relevant photographs so we can investigate.

Returns and custom work

Because parts are made to customer-supplied specifications, returns are not handled in the same way as standard retail goods. If there is a problem with the order, customers should contact us directly so the issue can be reviewed against the original file, order details, and supplied specification.
