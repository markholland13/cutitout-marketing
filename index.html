Frequently Asked Questions

the main questions customers ask before they upload a file

These stay on the homepage because they remove hesitation, but the main action is still to move the customer toward the quote page.

What file types do you accept?

The quote flow is built around DXF files. Start with a clean DXF and check the file guidance before upload.

What materials can I quote online?

Common online options include mild steel, stainless steel, aluminium, and acrylic, with material and thickness selected during quoting.

Can I order one-offs as well as batches?

Yes. The service is set up for one-off parts, repeat jobs, and small production runs, with quantity affecting the price per part.

Can I add powder coating?

Yes, where supported. Finishing can be added during quoting so the total price reflects both cutting and coating requirements.

Do you deliver across the UK?

Yes, the service is aimed at UK mainland delivery, and shipping is included as part of the online order flow.

Can I reorder a part later?

Yes. Previous jobs can be found again through your account, which makes repeat ordering much quicker.
